# 12ヶ月キャッシュフローシミュレーター

## 概要
自賠責保険の請求・入金サイクルを考慮した、12ヶ月間のキャッシュフロー予測ツールです。

## 使い方

### インストール不要
1. `index.html` をブラウザで開くだけで動作します
2. インターネット接続が必要です（Chart.jsライブラリをCDNから読み込むため）

### 基本機能

#### 1. 基本設定
- **月初手元現金**: シミュレーション開始時の手元資金
- **月次通常収入**: 毎月の固定収入（自賠責以外）
- **月次支出**: 毎月の固定支出
- **入金サイクル**: 自賠責請求から入金までの期間（1〜4ヶ月）

#### 2. 過去請求の設定
- **3月請求**: 5月に入金される3月分の請求額
- **4月請求**: 5月に入金される4月分の請求額
- サイクル2ヶ月の場合、5月に3月・4月分がまとめて入金されます

#### 3. 月次請求額の設定
- 5月〜翌4月までの各月の自賠責請求額を個別に設定できます
- 数値を変更すると即座にグラフと表が更新されます

#### 4. 入金フロー確認
- 請求月と入金月の対応関係が視覚的に確認できます
- 緑色: 請求月（金額表示）
- 青色: 入金月（まとめ入金額表示）
- 白色: 入金なし

### 確認できる指標

#### サマリー指標
- **12ヶ月後の手元現金**: 期末残高
- **月平均収支差**: 月ごとの平均収支
- **最低残高**: 12ヶ月間で最も残高が低くなる金額
- **要注意月数**: 残高が50万円を下回る月の数

#### グラフ
- 青い折れ線: 月末の手元現金残高
- 棒グラフ: 月次収入（青=自賠責入金あり、薄赤=入金なし）
- 赤い点: 残高50万円未満の警告

#### 詳細表
- 各月の期首残高、収入、支出、期末残高を確認できます
- 自賠責入金がある月は背景が薄青色になります

## 入金ロジックの説明

### 例: サイクル2ヶ月の場合
- **5月入金**: 3月・4月の請求額合計
- **7月入金**: 5月・6月の請求額合計
- **9月入金**: 7月・8月の請求額合計
- **11月入金**: 9月・10月の請求額合計
- **1月入金**: 11月・12月の請求額合計
- **3月入金**: 1月・2月の請求額合計

### サイクルを変更した場合
- サイクル1ヶ月: 毎月入金
- サイクル3ヶ月: 3ヶ月分まとめて入金
- サイクル4ヶ月: 4ヶ月分まとめて入金

## カスタマイズ

### デフォルト値の変更
`index.html` の以下の箇所を編集してください:

```javascript
// 初期値の設定
const DEFAULT_BILLING = [430000,430000,430000,430000,430000,430000,430000,430000,430000,430000,250000,250000];
```

### 警告閾値の変更
残高50万円未満で警告が出る設定を変更する場合:

```javascript
if(cash < 500000) dangerCount++;  // この数値を変更
```
